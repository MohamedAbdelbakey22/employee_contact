# Employee Contact Information Module

## Overview
This Odoo 17 module is designed to manage basic employee contact information. It allows users to perform standard CRUD operations (Create, Read, Update, Delete) on employee records.

## Features
- Add Contacts: Easily add new employees with their first name, last name, and email.
- List View: View all employee contact information in an organized list.
- Update: Edit existing employee details directly through the form view.
- Delete: Remove employee records when they are no longer needed.

## Technical Requirements
- **Odoo Version**: 17.0
- Dependencies: `base` module.

## Installation & Setup
1. Extract the `employee_contact` folder into your Odoo `custom_addons` directory.
2. Ensure the path is added to your `odoo.conf` file.
3. Restart your Odoo server.
4. Activate **Developer Mode**.
5. Go to **Apps** -> **Update Apps List**.
6. Search for "Employee Contact" and click **Install**.
